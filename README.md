# BatchFFmpeg 批量重命名与格式转换工具

## 简介
BatchFFmpeg 是一个用于批量重命名文件并自动将 `.ts` 视频文件转换为 `.mp4` 格式的实用脚本。支持多种排序方式，适合整理剧集、课程等批量视频文件。

## 功能
- 批量重命名文件为“第N集”格式，可自定义前缀
- 支持按文件名、大小、修改日期、括号内数字排序
- 自动将 `.ts` 文件转换为 `.mp4`（调用 ffmpeg），其他文件直接复制
- 测试模式下仅打印排序结果，不做实际操作

## 依赖
- Python 3.6 及以上
- 需安装 [ffmpeg](https://ffmpeg.org/) 并配置到系统环境变量（仅处理 .ts 文件时需要）

## 使用方法

### 命令行参数
| 参数         | 说明                         | 是否必需 |
|--------------|------------------------------|----------|
| -i, --input  | 输入目录                     | 必需     |
| -o, --output | 输出目录                     | 必需     |
| -p, --prefix | 输出文件名前缀（可选）        | 否       |
| -s, --sort   | 排序方式 name/size/date/duplicate | 否（默认 name） |
| -t, --test   | 测试模式，仅打印不操作        | 否       |

### 示例
```bash
python main.py -i "源文件夹路径" -o "目标文件夹路径" -p "我的剧集" -s duplicate
```

- 按文件名排序，重命名为“我的剧集 第001集.xxx”格式
- 若为 `.ts` 文件，自动转为 `.mp4`，否则直接复制

#### 测试模式
```bash
python main.py -i "源文件夹路径" -o "目标文件夹路径" -t
```
仅打印排序后的文件列表，不做实际拷贝/转换。

## 排序方式说明
- `name`：按文件名字母顺序
- `size`：按文件大小升序
- `date`：按最后修改时间升序
- `duplicate`：按文件名括号内数字排序（如“xxx(12).ts”）

## 文件结构
- `main.py`：主程序，命令行入口
- `FileList.py`：文件列表与排序逻辑

## 注意事项
- 输出目录会自动创建
- 文件名中会自动补齐三位数字（如“第001集”）
- `.ts` 文件转换需 ffmpeg 支持
